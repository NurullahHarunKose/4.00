## 📖 Proje Hakkında

Öğrencilik hayatının en büyük streslerinden biri notları ve ortalamayı takip etmektir. **4.00**, öğrencilerin "Bu dersten CC alırsam ortalamam kaç olur?" veya "Bursumun kesilmemesi için bu dönem en az kaç yapmalıyım?" gibi kritik sorularına saniyeler içinde cevap veren gelişmiş bir web/mobil uygulamasıdır. 

Karmaşık Excel tabloları veya formüllerle uğraşmak yerine, sezgisel ve modern arayüz sayesinde akademik gidişatınızı kolayca analiz edebilirsiniz.

---

## 🌟 Özellikler

- **Çoklu Dönem Desteği:** Sadece mevcut dönemi değil, geçmiş dönemleri de kaydederek kümülatif bir hesaplama yapabilme.
- **Dinamik GANO ve YANO Hesaplama:** Ders kredisi ve harf notunu girdiğiniz an ortalamanızın anlık olarak güncellenmesi.
- **Hedef Ortalaması Simülatörü:** Mezuniyet veya burs hedefiniz için kalan kredilerinizle hangi notları almanız gerektiğini hesaplayan akıllı algoritma.
- **Kişiselleştirilebilir Harf Sistemi:** Farklı üniversitelerin farklı harf-kredi sistemlerine (AA/BA/BB veya A1/A2/B1 vs.) tam uyumluluk sağlama ayarları.
- **Yerel Depolama (Local Storage):** Girdiğiniz verilerin tarayıcınızda güvenli bir şekilde saklanması; uygulamayı kapatsanız bile verilerinizin silinmemesi.
- **Karanlık Mod (Dark Mode):** Göz yormayan, şık gece modu seçeneği.
- **Mobil Uyumlu (Responsive):** Telefondan, tabletten veya bilgisayardan sorunsuz kullanım.

## 🚀 Başlarken (Kurulum)

Projeyi kendi bilgisayarınızda derlemek ve geliştirmek için aşağıdaki adımları sırasıyla uygulayın.

### Gereksinimler

- [Node.js](https://nodejs.org/en/) (v16.0 veya üstü)
- npm veya yarn (Node.js ile birlikte gelir)
- Git

### Yerel Kurulum

1. Repoyu bilgisayarınıza klonlayın:
   ```bash
   git clone https://github.com/NurullahHarunKose/4.00.git
   ```

2. Proje klasörüne geçiş yapın:
   ```bash
   cd /NurullahHarunKose/4.00
   ```

3. Gerekli kütüphane ve paketleri yükleyin:
   ```bash
   npm install
   # veya
   yarn install
   ```

4. Çevresel değişkenleri (Environment Variables) ayarlayın:
   Proje ana dizininde bulunan `.env.example` dosyasını kopyalayarak adını `.env` yapın ve gerekli alanları doldurun.
   ```bash
   cp .env.example .env
   ```

5. Geliştirme (Development) sunucusunu başlatın:
   ```bash
   npm run dev
   # veya
   yarn dev
   ```

Uygulama başarıyla başlatıldığında tarayıcınızdan `http://localhost:3000` adresine giderek projeyi görüntüleyebilirsiniz.

## 💡 Kullanım Senaryoları

1. **Ders Ekleme:** Ana sayfadaki forma dersin adını (örn: "Algoritmalara Giriş"), kredisini (örn: "4") ve beklediğiniz harf notunu (örn: "BA") girin. "+ Ekle" butonuna basın.
2. **Tabloyu Yönetme:** Eklediğiniz dersler listede görünecektir. Listeden ders düzenleyebilir veya yanlış eklenenleriilebilirsiniz.
3. **Genel Ortalama Hesabı:** Ekranın sağ (veya üst) panelindeki mevcut tamamlanmış kredi ve mevcut ortalama (GANO) alanlarını doldurarak, bu dönemin genel gidişatınıza etkisini gözlemleyin.
